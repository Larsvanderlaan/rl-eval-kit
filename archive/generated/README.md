# Generated Archive

This directory holds generated or bulky pre-migration material moved out of the
active source layout. It is intentionally ignored by git.

Retained categories:

- pre-migration full `FQE_neurips`, `FQE_calibration_neurips`, and
  `hopper_fqe_benchmark` trees with broad result forests;
- top-level generated `outputs/`;
- pre-migration exploratory experiment snapshots and figures;
- pre-migration full `IRL_neurips` tree, legacy IRL paper copy, and IRL
  simulation outputs;
- Python and test caches;
- LaTeX build products moved out of canonical paper directories.

The active reproducible sources live under `experiments/neurips_bellman`,
`papers/neurips_bellman`, and `submission_bundles/neurips_bellman`.
