# IRL Papers

This directory contains the active inverse-reinforcement-learning paper sources.

- `journal_debiased_irl/`: JASA-style journal manuscript
  "Efficient Inference for Inverse Reinforcement Learning and Dynamic Discrete
  Choice Models" plus its appendix and bibliography.
- `conference_genpqr/`: NeurIPS-style GenPQR conference manuscript
  "Modular Inverse Reinforcement Learning via Policy Estimation and
  Q-Evaluation" plus paper-visible figures, supplementary snippets, and the
  last copied draft PDF.

Generated LaTeX build products stay out of git; curated source TeX, BibTeX,
paper-visible figures, and final copied PDFs are trackable.
