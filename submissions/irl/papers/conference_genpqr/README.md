# GenPQR Conference Paper

Active files:

- `main_neurips.tex`: NeurIPS-style manuscript copied from
  `main_neurips-3.tex`.
- `references.bib`: portable copy of the shared IRL bibliography.
- `neurips_2026.sty`: local style file so the paper can compile from this
  directory.
- `exp1_matched_curve.png`, `exp2_method_comparison.png`,
  `many_action_reward_mse.png`, and `many_action_anchor_count.png`: figures
  referenced by the manuscript.
- `supplement/`: paper snippets and appendix material retained for audit.
- `paper_draft.pdf`: copied reference draft.

Experiment code lives in `submissions/irl/experiments/conference_genpqr/repro`.
