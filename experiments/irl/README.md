# IRL Experiments

Active IRL experiment code is split by paper.

- `journal_debiased_irl/`: JRSSB/JASA debiased-IRL simulation runner and
  reusable simulation module. Defaults write caches and outputs under `/tmp`.
- `conference_genpqr/`: compact GenPQR/DeepPQR reproducibility code copied
  from the paper-facing `IRL_neurips_paper_repro` bundle.

The old broad `IRL_neurips` tree, local virtual environment, exploratory
notebooks, outputs, and caches were moved to `archive/generated/irl_pre_migration`.
