# IRL GenPQR NeurIPS Reproducibility Folder

This folder contains the compact code and paper-visible artifacts needed to
reproduce the GenPQR/DeepPQR experiments in the NeurIPS paper draft. It omits
raw run outputs, exploratory scripts, notebooks, old drafts, virtual
environments, caches, and related-work material.

## Contents

- `data_generation.py`, `policy_estimation.py`, `q_evaluation.py`,
  `reward_recovery.py`, `utils.py`, `deeppqr_baseline.py`, and
  `lightgbm_fqe.py`: core GenPQR/DeepPQR implementation.
- `experiments/run_paper_experiments.py`: reproduces the main matched
  DeepPQR-vs-GenPQR comparison and the high-sample method comparison.
- `experiments/action_scaling_runner.py`: reproduces the many-action appendix
  figure.
- `paper_artifacts/`: the four external graphics included in the NeurIPS
  paper source.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Reproduce The Paper Experiments

From this folder, run:

```bash
python experiments/run_paper_experiments.py --replicates 100 --jobs 4 --output-dir outputs/paper_run_100
python experiments/action_scaling_runner.py --replicates 2 --jobs 4 --output-dir outputs/many_action_fixed_sample_quick
```

The scripts write raw CSVs, summaries, and regenerated plots under `outputs/`.
Those generated files are useful for auditing a run, but they are not part of
the distributed paper artifact set.

## Paper Artifact Map

The NeurIPS source `main_neurips.tex` externally includes exactly these files:

- Main matched comparison: `paper_artifacts/exp1_matched_curve.png`.
- Appendix high-sample method comparison:
  `paper_artifacts/exp2_method_comparison.png`.
- Appendix many-action panels:
  `paper_artifacts/many_action_reward_mse.png` and
  `paper_artifacts/many_action_anchor_count.png`.

The main matched comparison table and high-sample method table are embedded in
the paper TeX. They are regenerated by `run_paper_experiments.py` as CSV and
Markdown outputs under the chosen `outputs/` directory.

## Quick Verification

For a one-seed smoke check:

```bash
python experiments/run_paper_experiments.py --replicates 1 --jobs 1 --output-dir /tmp/irl_genpqr_repro_main_check
python experiments/action_scaling_runner.py --replicates 1 --jobs 1 --output-dir /tmp/irl_genpqr_repro_action_check
```
